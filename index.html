Lastify

Love, tag, ban & playlist on Last.fm while you listen in Spotify on Mac OS X

How do I install it?

  1. Download and install SIMBL.
  2. Download the latest version of Lastify from the downloads page.
  3. Put the Lastify.bundle ﬁle into your /Users/your-username-here/Library/Application Support/SIMBL/Plugins directory.

If you need to uninstall Lastify for any reason, just delete the Lastify.bundle ﬁle.

A few notes…

  • Lastify is open source: all of the code is on GitHub.
  • Spotify doesn't ofﬁcially support plugins. Lastify, and all other SIMBL or Input Manager based plugins, are slightly dangerous (they may cause Spotify to crash).
  • When a new version of Spotify comes out Lastify will stop working and you will need to download a new version. It may take a while to produce a new version of Lastify depending on what's changed in Spotify.
  • I don't know (or really want to know) enough about Windows to make a Windows version, sorry.
